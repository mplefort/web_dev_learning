What is CSS
 - Cascading Stylesheets
 - styling language (not programming)
 - extended with Sass/Less, variables and conditionals
 
 3 Methods for Adding CSS
  - Inline CSS: Direclty in HTML element (NO!)
  - Internal CSS: Using <style> tags within the HTML document (not best practice)
  - External CSS: Linking an external .css file (Best Practice)

  Syntax
  selector {property: value;}

  Colors in CSS

  - html5 colors = coral, rose
  - hexadeciam = #00ff00
  - rgb(0, 0, 255)

  Web Safe Fonts

  Standard for class and ID

Positioning
    - static - default in order
    - relative - rel to normal position
    - absolute - target position inside relative element
    - fixed to window no matter scroll position
    - initial - set to default value
    - inherit - from another element
